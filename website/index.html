Características Principales

DroidMouse combina tecnología avanzada con simplicidad de uso para ofrecer la mejor experiencia de control remoto de ratón.

Sensores de Alta Precisión

Utiliza el giroscopio y acelerómetro integrados de tu dispositivo Android para capturar movimientos con precisión milimétrica.

Filtro de Kalman

Algoritmo de filtrado avanzado que elimina ruido y vibraciones, proporcionando movimientos suaves y precisos del cursor.

Calibración Automática

Sistema de calibración inteligente que compensa la deriva del sensor y adapta la sensibilidad a tus preferencias.

Comunicación TCP/IP

Conexión estable y de baja latencia mediante sockets TCP/IP, compatible con cualquier red local.

Multiplataforma

Servidor Python multiplataforma que funciona en Windows, macOS y Linux. Control total del ratón del sistema.

Código Abierto

Proyecto completamente open source bajo licencia GPL3. Contribuye, modifica y distribuye libremente.

¿Cómo Funciona?

DroidMouse utiliza una arquitectura cliente-servidor moderna para proporcionar control remoto del ratón de manera eficiente y confiable.

1

Captura de Sensores

La app Android lee datos del giroscopio y acelerómetro a 50Hz

2

Procesamiento

Filtrado con Kalman y calibración automática de sensores

3

Transmisión

Envío de datos procesados al servidor via TCP/IP

4

Control

Movimiento del cursor del ratón en tiempo real
