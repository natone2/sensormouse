Guía para Contribuir

¿Quieres ayudar a mejorar DroidMouse? Aquí tienes todo lo necesario para colaborar.

¿Cómo contribuir?

  1. Haz un fork del repositorio en GitHub.
  2. Crea una rama descriptiva para tu mejora o corrección.
  3. Realiza tus cambios siguiendo las buenas prácticas del proyecto.
  4. Haz un pull request explicando claramente tu contribución.

Buenas Prácticas

  • Usa mensajes de commit claros y concisos.
  • Incluye pruebas o ejemplos si es posible.
  • Respeta el formato de código y las guías de estilo.
  • Documenta cualquier cambio relevante.

Reportar Bugs

  • Abre un issue detallando el problema, pasos para reproducirlo y entorno.
  • Adjunta logs, capturas o información relevante.

Solicitar Nuevas Funciones

  • Describe la funcionalidad deseada y su utilidad.
  • Explica cómo mejoraría la experiencia del usuario.

Contacto y Comunidad

  • Únete a la comunidad en GitHub para debatir ideas y recibir ayuda.
  • Contacto directo: soporte@droidmouse.com
