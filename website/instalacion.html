Guía de Instalación

Sigue estos pasos para instalar y configurar SensorMouse en tu PC y tu dispositivo Android.

Requisitos Previos

  • PC con Windows, Linux o macOS
  • Python 3.8 o superior
  • Teléfono Android (Android 7.0+ recomendado)
  • Red WiFi local compartida entre PC y móvil

Instalación del Servidor (PC)

  1. Descarga o clona el repositorio desde GitHub.
  2. Abre una terminal y navega a la carpeta server/.
  3. Instala las dependencias con:
    pip install -r requirements.txt
  4. Ejecuta el servidor:
    python main.py

Instalación de la App (Android)

  1. Descarga el archivo APK desde la sección Descargar.
  2. Copia el APK a tu teléfono e instálalo (puede requerir activar "Orígenes desconocidos").
  3. Abre la app y asegúrate de estar en la misma red WiFi que tu PC.

Primer Uso

  1. En la app, introduce la IP y puerto del servidor (mostrados en la terminal del PC).
  2. Pulsa Conectar y comienza a controlar el ratón con los sensores del móvil.

Solución de Problemas

  • Consulta la sección Solución de Problemas si tienes inconvenientes.
  • Para dudas frecuentes, revisa la FAQ.
