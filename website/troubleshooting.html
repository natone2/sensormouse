Solución de Problemas

Guía para resolver los problemas más comunes al usar SensorMouse.

Conexión

La app no detecta el servidor

  • Verifica que ambos dispositivos estén en la misma red WiFi.
  • Asegúrate de que el servidor esté ejecutándose y el puerto sea correcto.
  • Desactiva el firewall o permite el puerto en tu PC.

El puerto está ocupado

  • Prueba con otro puerto libre y actualiza la configuración en la app y el servidor.

Sensores y Movimiento

El cursor se mueve de forma errática

  • Recalibra los sensores desde la app.
  • Coloca el móvil sobre una superficie estable al conectar.

Latencia alta o movimientos lentos

  • Reduce la distancia al router WiFi.
  • Evita interferencias y redes saturadas.

Instalación y Dependencias

Error al instalar dependencias Python

  • Usa pip install -r requirements.txt en la carpeta server/.
  • Verifica que usas Python 3.8 o superior.

Problemas con la app en Android

  • Activa "Orígenes desconocidos" para instalar el APK.
  • Reinicia el móvil si la app no responde.

Otros Errores

El ratón no se mueve pero la conexión es correcta

  • Verifica permisos de control de ratón en el PC.
  • Reinicia el servidor y la app.

¿No encuentras tu problema?

  • Consulta la FAQ o abre una issue en GitHub.
